# IPSA MCP Toolkit

Connect **Claude AI** to **IPSA 3** power system software using the Model Context Protocol (MCP) — analyse power networks using plain English.

---

## Requirements

- **IPSA 3** — must be installed and licensed on your machine (provides Python 3.11 + PyIPSA)
- **Claude Desktop** — [download here](https://claude.ai/download)

---

## Installation

1. Download the package from the link below
2. Extract the ZIP to any folder
3. Double-click `setup.bat` and follow the prompt
4. Restart Claude Desktop
5. Go to **Settings → Developer** and confirm `ipsa-power-system` shows **RUNNING**
